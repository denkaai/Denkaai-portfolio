# Denkaai | Cyber Intelligence Analyst

A high-performance, neon cyber-themed portfolio for Denkaai (KanyiWanyiri), showcasing Digital Forensics, OSINT, Incident Response, and Network Analysis work with interactive case cards, tools, skills, achievements, and contact links.

## Highlights
- Orbitron/Share Tech Mono/Rajdhani typography with scanline and grid glow aesthetic
- Fixed sticky navigation with mobile toggle, smooth anchor scrolling, and active section highlighting
- Reveal-on-scroll animations with prefers-reduced-motion accessibility support
- Structured sections: Hero, Terminal-style About, Investigation Cases, Tools, Skills, Achievements, Contact
- Clickable Achievements linking to real profiles:
  - Hack The Box: https://app.hackthebox.com/profile/denkaai
  - TryHackMe: https://tryhackme.com/p/denkaai
  - Evidence Packaging: https://github.com/denkaai
- Contact shortcuts:
  - Email: contact.denkaai@gmail.com
  - GitHub: https://github.com/denkaai
  - LinkedIn: https://www.linkedin.com/in/denkaai

## What’s Inside
- `index.html` — complete single-file site with all styles, scripts, and content embedded for easy hosting

## Run Locally
Open the file directly:
1. Download or clone the repository
2. Double-click `index.html` or open with any modern browser

## Deploy (GitHub Pages)
1. Push `index.html` (and this README) to the `main` branch
2. In the repo: Settings → Pages → “Deploy from a branch”
3. Select Branch: `main`, Folder: `/` (root), then Save
4. Your site will publish at:
   - User site: `https://denkaai.github.io/` (if repo is `denkaai.github.io`)
   - Project site: `https://denkaai.github.io/Denkaai-portfolio/`
5. Optional: set a custom domain and enforce HTTPS in Pages settings
